# ГЛОССАРИЙ

<br />

`capture_method:`

<br />

В случае значение “HOLD” - платеж захолдируется и её необходимо будет подтвердить. Если “AUTO” (стоит по умолчанию) то списывается автоматически. Это ENUM - с двумя значениями выше.

<br />

`external_id:`

<br />

Дополнительный идентификатор в вашей системе. Используется чтобы соблюдалась [идемпотентность](https://ru.wikipedia.org/wiki/%D0%98%D0%B4%D0%B5%D0%BC%D0%BF%D0%BE%D1%82%D0%B5%D0%BD%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C#:~:text=.-,%D0%92%20%D0%B8%D0%BD%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%82%D0%B8%D0%BA%D0%B5,-%5B%D0%BF%D1%80%D0%B0%D0%B2%D0%B8%D1%82%D1%8C%20%7C).
